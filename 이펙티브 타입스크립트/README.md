### 어떤 책인가?

### 왜 읽는가?

- 3번의 프로젝트에서 사용했던 타입 스크립트. 왜 쓰는지도 알겠고 좋은 것도 알겠다. 그러나 그간의 프로젝트에서 타입 스크립트를 사용하는 방식에 별다른 변화는 일어나지 않았던 것 같았다.  
분명 뭔가 더 ‘잘’ 사용할 수 있는 방법이 있을 건데 말이다.
- 타입 스크립트에 대한 얕은 지식에 깊이를 더하자.
